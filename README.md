# Friend-verse

A Facebook-like social media platform built with React.

## Features

- 🔐 Login / Register
- 📝 Create Posts with likes & comments
- 👥 Add / Remove Friends
- 💬 Real-time Chat (Panda Chat)
- 👤 User Profile Page
- 🌐 Built using:
  - ReactJS
  - React Router
  - Context API
  - JSON Server (as fake backend)
  - CSS

## Installation

```bash
npm install
npm start

npx json-server --watch db.json --port 3000
